ARTIST BOOKING SYSTEM

Objective
The main objective of the Artist booking system is to manage the details artists desired by the client for their event and their availability.
It manages all the information related to the artist and the event details.
The project is totally built at administrative end and thus only the administrator is guaranteed the access. 
The purpose of the project is to build an application program to reduce the manual work for managing artists.

Database Schema
login_details: Sno, username, password.
book_now:  Sno, artist, location, genre, date, timing, budget.
messages: Sno, name, email, number, subject, message.
